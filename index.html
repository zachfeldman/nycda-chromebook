Welcome to the NYCDA Chromebook

Welcome to this fun Chromebook for developers. To access the Linux development environment installed on this machine, first try pressing CTRL + ALT + Right Arrow (F2). If this doesn't work, follow these steps:

  1. While in the Chrome web browser, press CTRL + ALT + T to open up a new crosh session.
  2. Type shell and then press enter.
  3. Type sudo startunity
  4. You should be in the pre-setup Ubuntu desktop at this point! If not, try pressing CTROL + ALT + Right Arrow (F2).
